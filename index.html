FEATURES

4つの画面で、「行きたい」をかたちにする。

見つける、決める、出かける、ふり返る。OurPinsは下部のタブで、友達との予定をやさしくつなぎます。

01 Map

みんなの行きたいが、地図でわかる

友達が「行きたい」と思った場所が、ひとつの地図に集まります。カフェ・ご飯・お出かけ先をカテゴリ別にピンして、メモや写真も一緒に。誰がどこに行きたいか、ひと目で。

  • カテゴリ別にピン（カフェ／和食／買い物 …）
  • メモや写真を添えて登録できる
  • グループ全員の「行きたい」が1枚の地図に
02 Spots

人気スポットを、すぐ発見

みんなの反応が集まる場所をランキングで表示。Hot / New / Nearby / 行った、の切り口で、いまグループで盛り上がっているスポットがわかります。

  • 反応の集まる場所をランキング表示
  • Hot・New・近く・行った で絞り込み
  • 友達のあいだで人気の場所に出会える
03 Plan

候補から、予定が決まる

候補日を出し合って投票。みんなの空き時間をもとに、行ける日を提案します。決まった予定は、そのまま端末のカレンダーへ。

  • 候補日を出し合って投票で決定
  • 空き時間から行ける日を提案
  • 端末のカレンダーと連携
04 Memories

行った場所が、思い出になる

行った場所は、写真とひとことで思い出に。グループのアルバムとして、これまでの履歴が自然に残っていきます。

  • 写真＋ひとことで記録
  • グループのアルバムに自動でまとまる
  • 行った場所の履歴をふり返れる
HOW IT WORKS

3ステップで、出かけたくなる。

むずかしい設定はいりません。グループを作って、ピンして、出かけるだけ。

1

グループを作る

招待コードを送るだけで、友達やカップルを招待。ひとつの地図をみんなで共有できます。

2

行きたい場所をピン

気になるカフェやお店をピン。「行きたい / かなり / 行った」で、気軽に反応し合えます。

3

予定化して思い出に

候補から予定を決めて、出かけたら写真で思い出に。追加や反応はリアルタイムで全員に同期。

MORE

友達同士で、気持ちよく使える。

日々の「ちょっと行きたい」を、無理なく共有できる小さな工夫。

リアクション

「行きたい / かなり / 行った」で気持ちを共有。誰が押したかもわかります。

招待コード

コードを送るだけでグループに参加。ひとつの地図をすぐに共有できます。

リアルタイム同期

追加も反応も、その場で全員の画面に反映。離れていても、ひとつの地図を一緒に。

お知らせ通知

新しいピンや、予定が決まったときにプッシュでお知らせ。見逃しません。

グループ内だけで共有

共有はメンバーの中だけ。場所も思い出も、外部には公開されません。

メモ＆写真

ピンにメモや写真を添えて、「行きたい理由」も一緒に残せます。

アカウントの削除について

OurPinsアカウントは、アプリ内からいつでも削除できます。本ページでは、削除の手順と、削除されるデータについてご案内します。

OurPinsアカウントは、アプリ内からいつでも削除できます。以下の手順でお進みください。

削除の手順

  1. OurPinsアプリを開きます。
  2. 画面右上のプロフィール（アバター）を開きます。
  3. 「アカウント」セクションの「アカウント削除」を選択します。
  4. 確認ダイアログで、削除を確定します。

削除される主なデータ

  • アカウント情報・プロフィール（名前・アバター・写真）
  • 追加した場所（ピン／スポット）、リアクション、コメント
  • 候補日・投票、決定した予定
  • 思い出（写真を含む）、通知設定 など

ご注意

法令の遵守、不正利用の防止、セキュリティ対応のため、一部のデータを一定期間保持する場合があります。詳細はプライバシーポリシーをご確認ください。

なお、グループに共有済みの一部コンテンツは、サービスの性質上、他のメンバーの環境に残る場合があります。

削除できない場合

うまく削除できない場合は、support@ourpins.app までお問い合わせください。

プライバシーポリシー お問い合わせ

友達との「ここ行こう」を忘れず、反応 → 予定 → 思い出までつなげる共同マップアプリ。
